## 概述

基于容器技术的软件配置管理系统
想更多得了解项目可以看一下文件中的论文。


## 主要功能模块

### alchimest
镜像（IMAGE），组件（COMPONENT），软件包（PACKAGE）的配置管理。

### furion
集群（REGION），服务（SERVICE），环境（ENVIRONMENT）的配置管理。

### coil
记录alchimest和furion的地址，将软件包和环境相连（Link，或称“发版”），记录发版行为，并提供结果的下载。
